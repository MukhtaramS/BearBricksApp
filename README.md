# BearBrick Identifier

A Swift-based machine learning project where users can explore different BearBrick types, learn about collaborations, and identify a BearBrick directly inside the app by taking a photo or pasting an image.

**Features:**  

- Browse BearBrick categories with info on designs, styles, and collaborations

- Identify a BearBrick by snapping a photo or uploading an image

- Custom-trained ML model with high accuracy on a small curated dataset

- Simple, clean interface for quick discovery
